Simplification of OSM Admin Boundaries
======================================

Dependencies
------------

* libosmium-dev and its dependencies except GDAL and GEOS
* cmake
* C++ compiler with C++11 support

Building
--------

    mkdir build
    cd build
    cmake ..
    make

Unit Tests
----------

Build the project as described above, afterwards run `make test`.
